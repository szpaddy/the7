Unregister, remove or disable enojis from wordpress. It also unrigister the emojis from the wordpress post editor, makes your site fast and free by removing non usable javascript files.

== Description ==
The plugin size is less than 2KB. You have to just install it and it will do all its tasks in the background. It will helps to disable or remove emojis from your wordpress websites without any hassle.

**How it works**

It uses simple php code snippets to unregister the emojis from your wordpress site. It also unregister the emojis support form the post editor.

== Installation ==
1. Upload the `remove-emojis` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. That's it!
